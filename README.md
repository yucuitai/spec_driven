# Spec-Driven - 交互式开发文档生成工具

一个交互式CLI工具，帮助开发者从产品描述到完整的项目文档（Spec、Phase、AGENTS.md）

## 特性

- 交互式向导，逐步收集产品信息
- 自动生成 Spec 文档（产品概述、功能列表、页面/视图、数据模型、用户流程）
- Phase 阶段拆分与验收计划
- 支持多种AI工具（Claude、Codex）配置
- 历史版本管理

## 安装

```bash
npm install
```

## 使用

```bash
npm run dev
```

## 开发阶段

- Phase 0: 项目初始化 ✅
- Phase 1: CLI基础框架
- Phase 2: 规划与确定规格
- Phase 3: 基础实现
- Phase 4: Phase拆分
- Phase 5: AGENTS/CLAUDE/CODEX模板
- Phase 6: 历史版本导出
- Phase 7: AI工具集成
- Phase 8: 上线部署
